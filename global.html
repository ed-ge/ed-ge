main(gameObjects, gameBehaviors, scenes, options)

Main function for the game. This functon takes in game-specific game objects, behaviors, and scenes and starts the game loop.

It also accepts an optional "options" parameter which can override the default behavior.

The options object can have the following key value pairs

  • startScene:{String} overrive the start scene provided in the scenes object
  • runUpdate:{bool} if true, prevents update from being called. The Engine wil only render the initial state.
Parameters:
Name Type Description
gameObjects Array

An array of game objects that can be in the game
gameBehaviors Array

An array of behaviors that can be in the game
scenes Object

An object specifying the start scenes and scene definitions
options Object

An object with options that override the defaults
